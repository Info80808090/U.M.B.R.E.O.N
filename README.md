# U.M.B.R.E.O.N

A Simple Client



## Features

- Web Cloner
- Password generator
- Pokemon Search
- Tiny Url

## Installation Guide :D

1. Clone this repository:
   ```
   git clone https://github.com/Info80808090/Umbreon-js
   ```
2. Navigate to the project directory:
   ```
   cd Umbreon-js
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Run the application:
   ```
   node index.js
   ```
